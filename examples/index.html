From Wikipedia:

Keyhole Markup Language (KML) is an XML file format for expressing geographic annotation and visualization within Internet-based, 2D maps and 3D Earth browsers.

The KML file specifies a set of features (placemarks, images, polygons, 3D models, textual descriptions, etc.) for display in Google Earth or any other geospatial software implementing the KML encoding. Each place always has a longitude and a latitude. Other data can make the view more specific, such as tilt, heading, altitude, which together define a "camera view".

KML files are very often distributed in KMZ files, which are zipped files with a .kmz extension. The contents of a KMZ file are a single root KML document and optionally any overlays, images, icons, and COLLADA 3D models referenced in the KML including network-linked KML files.

kml_01.ncl (example01.kml): Demonstrates how to open a KML document, add Placemark and Point elements, and write a KML document to a file.
kml_02.ncl: Demonstrates how to add Polygon elements to a KML document and style them with the PolyStyle element.
kml_03.ncl: Demonstrates how to add a NetworkLink element to a KML document to reference another remote KML document.
kml_04.ncl: Demonstrates how to add 3D COLLADA models to a KML document and how to use the LookAt element.
kml_05.ncl: Demonstrates how to crop the map and label bar from an NCL plot created with gsn_csm_contour_map_ce and add them to a KML document as GroundOverlay and ScreenOverlay elements.
kml_06.ncl: Same as the example above but shows how to make regions transparent by setting the resource kmlCropAlphaColor to the same color as resource mpFillColors.
kml_07.ncl: This example script contributed by Aditya R. Kartadikaria shows how to animate multiple GroundOverlay elements using the TimeSpan element.
